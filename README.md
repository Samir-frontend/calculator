## 🧮 Overview

> A clean, modern **calculator app** that performs basic arithmetic operations with a sleek dark UI and smooth button interactions — built purely with vanilla JavaScript logic.

---

## ➕ Operations Supported

<div align="center">

| ➕ Add | ➖ Subtract | ✖️ Multiply | ➗ Divide | 🔄 Clear |
|:---:|:---:|:---:|:---:|:---:|
| Real-time | Real-time | Real-time | Real-time | Reset All |

</div>

---

## ✨ Features

- 🖥️ **Live Display** — shows expression as you type
- 🎹 **Button Click Feedback** — smooth press animations
- 🔁 **Clear (C) / All Clear (AC)** functionality
- 🧠 **Operator Precedence Handling** in calculations
- 📱 Fully responsive — works great on mobile too

---

## ⚙️ How It Works

```javascript
1. User clicks number/operator buttons
2. Display updates in real-time
3. On "=" → JavaScript evaluates the expression
4. Result is shown instantly on screen
```
